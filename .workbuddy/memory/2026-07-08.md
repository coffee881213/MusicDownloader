# 2026-07-08 Work Log

## YouTube/Bandcamp video format download fix
- **Problem**: yt-dlp format string included `/best` fallback which downloads video+audio combined formats (.mp4/.webm), wasting bandwidth and leaving useless video files
- **Fix 1**: Removed `/best` from format strings:
  - YouTube: `"bestaudio[ext=flac]/bestaudio[acodec=flac]/bestaudio"` (was `/bestaudio/best`)
  - Bandcamp: `"bestaudio[ext=flac]/bestaudio"` (was `/bestaudio/best`)
- **Fix 2**: Changed fallback format from `"best"` (video+audio) to `"bestaudio"` (audio-only)
- **Fix 3**: Added `VIDEO_EXTS` constant: `.mp4/.mkv/.avi/.mov/.wmv/.flv/.ts/.3gp/.webm/.m4v/.mpg/.mpeg/.vob`
- **Fix 4**: Added `_cleanup_video_in_dir()` function — deletes video-format files left by yt-dlp after download or on error
- **Fix 5**: Video cleanup called after every download attempt and error in both YouTube and Bandcamp functions
- Synced download_music.py to original project directory
